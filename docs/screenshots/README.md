## Recommended Manual Screenshot Checklist

For a cleaner long-term naming convention, consider adding or replacing screenshots with:

- [ ] `home.png`
- [ ] `login.png`
- [ ] `lobby.png`
- [ ] `room-created.png`
- [ ] `game-room-two-players.png`
- [ ] `game-over-or-rematch.png`

Capture screenshots from the deployed app after confirming the backend is awake. Avoid including real passwords, personal email addresses, production database details, or JWT values.
