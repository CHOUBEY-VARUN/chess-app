# ChessRoom Client

React + Vite frontend for ChessRoom.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run preview
```

## Environment

Create `.env` from `.env.example`.

```env
VITE_API_BASE_URL=http://localhost:3000
```

For deployment, set `VITE_API_BASE_URL` to the hosted backend URL.
